Ornstein-Uhlenbeck models for Phylogenetic Comparative Hypotheses

See the [package website](https://kingaa.github.io/ouch/).

#### Binary install

```
install.packages("ouch",repos="https://kingaa.github.io/")
```

#### Source install

```
library(devtools)  
install_github("kingaa/ouch")
```
